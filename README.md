# shareable

Public surface for one piece of Andrew Hjelle's working notes.

**[`index.html`](index.html)** — the **Style Guide** for the Beauty Co Daily atelier system.

Seven atelier styles (Sage, Reactor, Aperture, Ukiyo, Mural, Terrazzo, Dunes) that re-skin the same modular content kit each day. The page includes a live "Today's Dials" composer — palette · type · geometry · texture · motion · layout · mood — that lets you build a custom skin from the same vocabulary.

Self-contained single HTML file.
