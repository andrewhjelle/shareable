# shareable

A small public surface for Andrew Hjelle's working notes.

- **[`index.html`](index.html)** — the **Knowledge Atlas**: a daily-updated map of AI · World & Markets · Beauty & Retail. Built from primary sources (Anthropic, OpenAI, BoF, WWD, etc.) with verbatim quotes and explicit honesty marks; aimed at being something you'd actually want to read.
- **[`style-guide.html`](style-guide.html)** — the design system behind the brief: seven atelier styles (Sage, Reactor, Aperture, Ukiyo, Mural, Terrazzo, Dunes), a composer that re-skins the same content kit each day, and a live "Today's Dials" composer.

Updated daily via the `atlas-update` skill. Source content lives in a private repo; this surface ships the rendered HTML.
